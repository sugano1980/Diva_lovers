　正騎士試験編1.01ver以降であればデータの引き継ぎできます。
  DL版の場合は、プレイした時にゲームのフォルダ内に生成された全ての.savデータを、新しく配信されたゲームのフォルダ内に移し替えてください。  diva_lovers_1_sf.sav ,diva_lovers_1_tyrano_data.sav,diva_lovers_1_tyrano_quick_save.sav等と名前が付いていると思います。
  ブラウザ版の場合は、引き継ぎは必要ありません。よりスムーズなプレイのためにはDL版をお勧めしています。


操作方法、説明など詳細は下記をご覧ください。



※ゲームの始め方
「Diva_Lovers_2」フォルダの中のDiva_lovers_2.exeというアプリケーションを起動してください。
容量があるため起動に多少時間がかかる場合があります。

※ゲームの手順。

タイトル画面にて、
 新しくゲームをはじめる→New　Game
途中からはじめる→Load Game
各種設定→Config(文字種の変更はここのコンフィグ画面でしか出来ません。最初に設定しておくことをお勧めします）
プレイした直前の状態から始める（フリーズしてしまった場合など）→自動保存データの読み込み

 ※ゲームは大きくわけて物語部分と自由行動部分にわかれます。

●物語部分
 
 背景と、文章が表示されます。

 ※左上の主人公アイコン→クリックすると、セーブロード、タイトルのボタンが出ます。
 戻るときには右上に移動した同じアイコンをクリックすることで戻れます。

※上部真ん中のボタン４種
Jump→次の選択肢に飛びます。移動がある場合にはMap画面に飛びます。
 初回プレイ時も利用できますが、誤って押さないようにご注意ください

Config→音量、スキップ機能など各種の設定ができます。
              Live2d表示中は使えないことがあります。

Skip→クリックで早送りします。
Log（本のアイコン）→これまで読んだ文章を確認できます。


●自由行動部分
 

 外出→王都の散策ができます。MAP上で一日に回れる場所は２箇所までです。
 イベントがある場合には「！」マークがつきます。赤い「！」はメインイベント。黄色の「！」はサブイベントです。

 稽古→剣の稽古をします。技術がランダムにアップします。
 
 課題→騎士として必要な知識を学びます。頭脳がランダムにアップします。

 休む→一日休息をとります。何もできませんが、次に稽古、課題コマンドを使用した時の上昇率が上がります。

 闘技場→闘技場でモンスターと戦います。モンスターの経験値にもよりますが倒すと技術、頭脳、魅力がアップします。
　　　　一度全モンスターを制覇するとトロフィーがもらえ、戦闘で経験値は得られなくなりますが好きな男性キャラと共闘することができます。
　　　　ただし、好感度等ステータスは上がりません。
 料理→エルムナード編から。厨房を借りて料理します。コマンドで料理した場合は自分のステータスが上がります。レシピ、追加材料、スパイスによって上昇率は変わります。
　　　
 占い→男性キャラのプロフィール、自分への好感度等を確認することができます。
　　　また、MAP画面でガイアの森に行くとガイアの巫女から好感度アップのコツなどが聞けます。
 
 アイテム→戦闘用アイテム、装備用アイテム、料理用アイテム、その他アイテムの確認ができます。その他アイテムで使用できるものは使うことができます。アイテムを１クリックで下に説明が表示されます。同時に表示されるゴミ箱をクリックで捨てることができます。元の画面に戻る場合にはアイテムボタンを再度クリックしてください。
 
 設定→BGM音量、SE音量、文字の大きさ、メッセージスピードの変更ができます。
 
 装備欄の変更ボタンを押すと、装備を変更することができます。
 装備の上段は髪型やアクセサリー、下段は服装です。変更すると主人公のモデルも変更されます。
 OKのボタンを押すと元の画面に戻ります。
 男性キャラの好みの服装をすると、何か言ってくれるかもしれません。

※行動画面でのMAPイベント通知
　MAP上でなくても外出先で何らかのイベントがある場合に表示されます。仲間がいると起きないイベントもあります。

※ロードについて
データロードの時、眠っている主人公アイコンが現れます。その後立ち上がったアイコンの表示が消えればロード完了です。

※選択肢について
選んだ選択肢により好感度があがります。
さがることはありません。
 好感度があがると、光と音とともに左上にアイコンが現れます。
 
※好感度について
 選択肢、服装や髪型、食事の際に選んだ店、メニューや会話によって上昇します。
 また、食事時、以前あったときよりもパラメータが大きく上昇しているとコメントと共に好感度が上がります。
（反応するパラメータは男性キャラにより異なります）
 男性キャラの好みを把握すると、効率的に好感度をあげられます。

※会話について
男性キャラとの会話話題を、一度選んだ後でも何度でも選択できるように仕様を変更しました。一度選ぶと灰色に色が変わりますが、選ぶことはできます。
時事的な話題のみ、イベントが終わった後に選択すると会話が変わっていることがあります。

※バトルについて
敵からの攻撃は、現れる盾のボタンをタイミングよく押すことで防げます。
また、男性キャラとの共闘の場合防御のコマンドがランダムで応援に変化します。応援すると、好感度と共に男性キャラのステータスがアップします。
戦いの絆がある場合、右上のデュアルゲージが現れ、応援でゲージを満タンにするとデュアル、強力な合体技を出すことができます。
また、戦いの絆があると、仲間から応援とステータス補助を受けられます。好感度により、デュアルの与えるダメージ、応援のステータス上昇率は変わります。

※デートについて
エルムナード編以降、マップに現れる男性キャラの居場所に行くと、デートに誘えます。
好感度と魅力により、断られる率が変わります。
好感度等の条件、またデート中の行動により、別れ際に男性キャラとスキンシップが取れます。
　
※Live2Dについて
 Live2Dをロードする場合、多少の時間がかかることがあります。（特にライオネス）
 ロード中の場合画面左上にLive2dLoad→LoadCompleteの表示が現れますので、止まってしまった場合などはご確認ください。

※データ読み込みについて
 ブラウザ版等からのプレイで画像表示などをスムーズにするため、タイトル画面の前（こちらは任意で選択できます）、ゲームの最初と、章の最初（カーニバル編）にデータ読み込みがあります。
　青いゲージがフルになるまでお待ちくださいませ。
　

※初期化のシステムについて
　負荷を軽減しフリーズを防ぐため定期的にリフレッシュしています。初期化は１日の行動の後、バトル前後、マップのイベント後、マップから部屋に戻った時、イベントの前等々。オートセーブも兼ねていますので、万が一フリーズしてしまった場合にはタイトル画面から「自動保存データの読み込み」ボタンを押すことで直前の状態に戻れますのでご活用ください。

※ブラウザ版について
 GoogleChrome等からのプレイの場合、初期化後に画面が止まる時があります。その場合はLoadComplete後にクリックすることで次に進みます。
